# Erica's OT Website

Theme derived from [Jekyll Serif](https://github.com/zerostaticthemes/jekyll-serif-theme).

## Getting Started

Get all dependencies
```sh
bundle
```

Serve
```sh
bundle exec jekyll serve
```
